    • Algorithms in C++, Parts 1 - 4: Fundamentals, Data Structures, Sorting, Searching by Sedgewick (3rd Ed., 1998).
    • Algorithms in C++, Part 5: Graph Algorithms by Sedgewick (3rd Ed., 1997).
    • Data Structures and Algorithms Using C++ by Akepogu and Palagiri (2010).
